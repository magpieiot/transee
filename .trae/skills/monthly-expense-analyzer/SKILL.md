# Monthly Expense Analyzer

This skill assists in analyzing monthly expense data.

## Core Responsibilities
1. **Data Parsing**: Parse and understand financial data (e.g., CSV, JSON, or text) provided by the user.
2. **Categorization**: Group expenses into logical categories (e.g., Housing, Food, Transport, Entertainment).
3. **Summary Generation**: Calculate total spending, identify the highest spending categories, and calculate average daily expenses.
4. **Insights**: Provide actionable insights or identify unusual spending patterns compared to standard benchmarks.

## How to use
- Ensure the user provides the expense data.
- Process the data systematically.
- Present the final analysis in a clear, readable format (like tables or bullet points).
